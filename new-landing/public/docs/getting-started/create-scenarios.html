Create Scenarios

Scenarios are how Genezio simulates real buyer questions. Good scenarios sound like the questions your customers actually ask AI assistants, including the constraints and follow-ups that influence which brands get recommended.

For each topic, Genezio generates scenarios across different intents, from open discovery questions to direct recommendation requests and brand-specific questions. You can review and edit these before confirming them.

Writing Effective Scenarios

  • Sound like a real buyer — draft prompts from real customer questions, call transcripts, and FAQs.
  • Include constraints — team size, budget, location, or technology stack all shape recommendations.
  • Cover branded and non-branded entry points — measure both how you are discovered and how you are evaluated by name.
  • Add follow-up turns — multi-turn paths reveal deeper model reasoning.

Changing a Scenario Type Later

You are not locked into your first choice. An existing scenario can be converted to another type later (for example, from a Prompter into a Recommender), and you choose the persona to use for the converted scenario.

To learn more about the available scenario types and when to use each, see Scenarios.
